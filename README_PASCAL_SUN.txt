SITE PASCAL SUN — V6 CMS

Cette version contient un vrai CMS via Decap CMS + Netlify Identity.

Adresse du site : /
Adresse admin : /admin/

IMPORTANT : pour que le CMS sauvegarde en ligne, il faut :
1. Mettre ce site dans un dépôt GitHub.
2. Le connecter à Netlify.
3. Dans Netlify : activer Identity.
4. Dans Netlify : activer Git Gateway.
5. T'inviter comme utilisateur admin avec ton email : sun.pascal09@gmail.com.
6. Aller sur https://ton-site.netlify.app/admin/

Ensuite tu peux :
- ajouter une œuvre,
- changer une photo,
- changer un prix,
- ajouter un lien PayPal,
- cliquer Publier.

Les visiteurs verront les changements après publication automatique Netlify.
